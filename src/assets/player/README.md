# Player Assets

Sprite for the player character.
Player radius is 14 px (diameter **28 × 28 px**).

| Asset file    | Dimensions | Description                                                            |
|---------------|------------|------------------------------------------------------------------------|
| `player.png`  | 28 × 28 px | Player character sprite, centered on the player's position (radius 14) |

If the file is missing or fails to load the renderer draws a white filled circle as fallback.
